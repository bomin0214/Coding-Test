## 풀이 설명
BFS(너비 우선 탐색)로 나이트의 최단 이동 횟수를 구한다. 나이트가 이동할 수 있는 8방향을 큐에 넣으며 탐색하고, 목표 칸에 처음 도달했을 때의 거리가 곧 최솟값이다. 시작과 목표가 같으면 0을 바로 반환하고, 체스판 범위를 벗어나거나 이미 방문한 칸은 건너뛴다.

---

## 시행착오
1. 시작 == 목표 예외 처리 누락
   BFS를 그냥 돌리면 시작점을 큐에 넣고 탐색을 시작하므로 (1,1) → (1,1) 같은 케이스에서 0이 아닌 잘못된 값이 나올 수 있다. 함수 진입 시 바로 return 0 처리가 필요하다.
2. 방문 배열 초기화를 -1로 안 하면 0과 구분 불가
   dist 배열을 0으로 초기화하면 시작점의 거리 0과 미방문 상태를 구분할 수 없어 탐색이 꼬인다. -1로 초기화해서 방문 여부를 명확히 구분해야 한다.
3. 테스트 케이스마다 dist 배열 재생성 안 함
   dist 배열을 전역으로 두고 초기화를 빠뜨리면 이전 케이스의 방문 정보가 남아 다음 케이스 결과가 틀린다. 매 테스트 케이스마다 새로 선언하거나 초기화해야 한다.
