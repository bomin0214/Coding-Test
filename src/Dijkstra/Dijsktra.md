## 풀이 설명
다익스트라 알고리즘을 사용해 1번 헛간에서 N번 헛간까지 가는 최소 비용을 구한다.  
우선순위 큐를 이용해 현재까지 비용이 가장 적은 헛간부터 탐색하며, 더 짧은 경로를 찾으면 거리를 갱신한다.

---

## 시행착오

1. BFS로 구현하려 했지만 가중치(소의 수)가 있어 최단 비용 보장이 되지 않았다.  
   → 다익스트라 알고리즘으로 변경.

2. 이미 더 짧은 경로가 있는 노드를 다시 탐색해 시간 초과가 발생했다.  
   →
```java
if (cur.dist > dist[cur.num]) continue;
```

조건 추가.

3. 양방향 길인데 한 방향만 저장해서 경로가 제대로 연결되지 않았다.  
   →
```java
graph[a].add(new Edge(b, c));
graph[b].add(new Edge(a, c));
```

처럼 양쪽 모두 저장.
